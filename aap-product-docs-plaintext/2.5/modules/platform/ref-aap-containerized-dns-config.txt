# Ansible Automation Platform containerized DNS configuration

To configure unbound DNS, see Chapter 2. Setting up an unbound DNS server - Red Hat Enterprise Linux 9.

To configure DNS using BIND, see Chapter 1. Setting up and configuring a BIND DNS server - Red Hat Enterprise Linux 9.

Optional: To ensure the installation program automatically picks up and applies your Ansible Automation Platform subscription manifest license, generate a manifest file. You can then download this manifest file for the installation program to refer to. To generate the manifest file, see 4.3. Obtaining a subscriptions manifest - Automation Controller User Guide.
