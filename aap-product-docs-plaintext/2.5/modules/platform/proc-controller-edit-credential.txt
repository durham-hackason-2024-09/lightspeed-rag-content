# Editing a credential

As part of the initial setup, you can leave the default Demo Credential as it is, and you can edit it later.

1. Edit the credential by using one of these methods:
* Go to the credential Details page and click btn:[Edit].
* From the navigation panel, select menu:Automation Execution[Infrastructure > Credentials]. Click btn:[Edit] next to the credential name and edit the appropriate details.
2. Save your changes.
