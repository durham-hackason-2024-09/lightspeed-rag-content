# Adding a team for a user

You can add a team for a user from the Users list view.

1. From the navigation panel, select menu:Access Management[Users].
2. Select the user that you want to which you want to add team membership.
3. Select the Teams tab to display the list of teams of which that user is a member.
4. Click btn:[Add Team(s)].
5. Select the check box for the team to which you want to add the user.
6. You can search this list by the team Name or Organization.

Until a team has been created and a user has been assigned to that team, the assigned team details for that user remain empty.
