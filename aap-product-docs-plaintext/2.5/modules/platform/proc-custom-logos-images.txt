# Setting a custom logo

Automation controller and Red Hat Ansible Automation Platform support the use of a custom logo.

You can add a custom logo by uploading an image and supplying a custom login message from the Platform gateway settings page.

1. From the navigation panel, select menu:Settings[Platform gateway].
2. In the Custom login info field, provide specific information (such as a legal notice or a disclaimer).
3. In the Custom logo field, provide an image file for setting up a custom logo (must be a data URL with a base64-encoded GIF, PNG, or JPEG image).
